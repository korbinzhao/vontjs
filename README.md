# Vont Framework

A full-stack framework combining Koa and React with file-based routing.

## Project Structure

- **vont/** - Core framework package (`@vont/core`)
- **demo/** - Example application demonstrating framework usage

## Quick Start

### Using in Demo

```bash
cd demo
npm install
npm run dev
```

### Framework Development

```bash
cd vont
npm install
npm run build
```
